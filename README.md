# ugv-ocr

This is an OCR module for the 2019 Canada UGV Competition.

The goal of this module is to extract text from images of pylons with objects with printed text.

## Usage

``` shell
cd scripts
sh ./scripts/start.sh
```

## Installation

``` shell
sh ./scripts/build.sh
```
